# Clarivo

Clarivo is a modern screen annotation tool built for lessons, tutorials, explanations, and presentations.

## Features

- Pen
- Highlighter
- Text
- Shapes
- Eraser
- Move and resize
- Screenshot capture
- Board mode
- Undo and redo

## Run on Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install PySide6
python3 main.py
